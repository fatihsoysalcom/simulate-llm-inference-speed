# Simulate LLM Inference Speed

This Python script simulates the token generation process of a Large Language Model (LLM) during inference. It demonstrates how computational delay per token impacts the overall 'tokens per second' metric, a critical factor for real-time LLM serving. The example highlights the performance challenges that specialized AI hardware aims to address by optimizing this per-token processing.

## Language

`python`

## How to Run

Save the code as `main.py`.
Then run from your terminal: `python main.py`
